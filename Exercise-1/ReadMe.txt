Simple framework for the cellular automaton in exercise 1.
Use "pip install -r requirements.txt" to install all required packages (python 3.10+).
Use "python main.py" to start the GUI.
Use --scenario to specify the scenario to run (e.g. "python main.py --scenario scenario_task1.json").
Use --algorithm to specify the algorithm to use (e.g. "python main.py --algorithm D"). 
Possible algorithms are "D" (Dijkstra), "F" (Fast Marching Method), and "S" is the default update rule.
Use --ignorePedestrians to ignore pedestrians (e.g. "python main.py --ignorePedestrians").
